## 기능 명세서

### 입력

- 자동차 이름
  - , 구분자로 구분
  - validation 호출
    - 이름 5자 이하
    - 차 0대인지 아닌지
- 시도 횟수
  - 숫자인지 아닌지
  - 음수인지 아닌지

### 출력

- 우승자 여러 명일 경우 , 로 구분

### 기능

- 랜덤 값 뽑기
- 전진 유무
